# *-Colors.txt
Color files for different mods to be used with Minetest MapServer and other Luanti tools

# Mapping tools
Supports the following mapping tools:
* https://github.com/minetest-tools/mapserver - Mapserver
* https://github.com/minetest/minetestmapper - Minetestmapper
* https://github.com/blockexchange/blockexchange - Blockexchange
* https://github.com/BuckarooBanzay/isogen - ISOgen
